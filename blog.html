What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents a web page as a tree structure, allowing developers to dynamically access, modify, and interact with elements using JavaScript.

How do you select an element from the DOM?

To select an element from the DOM, you can use JavaScript methods like:

  1. 1.getElementById
  2. 2.getElementsByClassName
  3. 3.getElementsByTagName
  4. 4.querySelector
  5. 5.querySelectorAll
What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.
